AngularJS: Pass data across controllers

5th October 2014


This blog assumes basic knowledge of AngularJS.

AngularJS controller allows you to attach a part of your DOM to particular scope. The scope holds data that your can use in the DOM that it is attached to. It is a good idea it create a separate controller for different parts of the DOM. This keeps the code clean, easy to maintain and debug. However, one may need to access a scope variable defined in one controller in some another controller. AngularJS services help solve this issue.
Here is how you would use a service to solve the issue. 

angular.module('app', [])
	.service('myService', function() {
		this.var1 = "tmp";
		this.var2 = "tmp";
	})
	.controller('ctrl1', ['$scope', 'myService', function ($scope, myservice) {
		$scope.myservice = myservice
		$scope.myservice.var1 = "This variable has been changed inside ctrl1 and is reflected on ctrl2";   
	}])
	.controller('ctrl2', ['$scope', 'myService', function ($scope, myservice) {
		$scope.myservice = myservice;
		myservice.var2 = "This variable has been changed inside ctrl2 and is reflected on ctrl1";
	}]);
